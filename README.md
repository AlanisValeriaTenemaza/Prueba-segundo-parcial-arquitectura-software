#  AndesFin – Microservicio de Inversiones
# Nombre Alanis Tenemaza

Microservicio en Node.js que permite simular inversiones óptimas respetando
el capital disponible del usuario.

## Tecnologías utilizadas
- Node.js
- Express
- PostgreSQL
- TypeORM
- Docker

## Endpoints
- GET /usuarios
- GET /productos
- POST /simulaciones
- GET /simulaciones/{usuarioId}
